# 合同风险识别 Skill

## 功能描述
自动识别各类合同中的风险条款，包括违约责任、付款条件、知识产权归属等关键风险点。

## 使用方法

```bash
qoder skill install legal-contract-risk-detection
```

## 输入
- 合同文本（支持PDF、Word、纯文本）

## 输出
- 风险条款列表
- 风险等级（高/中/低）
- 修改建议

## 示例

**输入：**
```
采购合同第5条：甲方应在收到货物后30天内付款，逾期按日千分之五支付违约金。
```

**输出：**
```
【高风险】付款期限过长
- 当前条款：30天付款期
- 风险：资金占用成本高
- 建议：缩短至15天或要求预付款

【中风险】违约金比例偏高
- 当前条款：日千分之五（年化182.5%）
- 风险：可能被认定为过高违约金
- 建议：调整至日千分之一至千分之三
```
