**在线访问：** https://james19890801.github.io/skill-hub

## 项目简介

CQ市场是一个基于流程管理体系构建的Skill Hub，帮助企业基于现有业务流程快速挖掘AI机会点，将最佳实践转化为可复用的AI能力（Skill）。

**核心理念：**
- 不是先建智能体，而是先挖Skill
- Skill是给AI看的SOP
- 基于流程体系构建AI能力
- 当天开发，当天就能用

## 使用方法

### 1. 浏览市场
访问 https://james19890801.github.io/skill-hub

### 2. 找到需要的Skill
- 按流程域浏览（IPD、供应链、人力资源等）
- 或使用搜索功能快速定位

### 3. 一键安装
点击Skill卡片上的"复制"按钮，粘贴命令到终端：

```bash
qoder skill install [skill-name]
```

## 什么是Skill？

Skill是给AI看的SOP——把原本给人看的操作流程，改写成AI能直接理解、执行、输出的原子能力单元。

**举个例子：**
- 原本：法务专员看SOP学合同审查 → 自己逐条审查 → 输出审查意见（2小时/份）
- 现在：调用「合同风险识别Skill」→ AI自动审查 → 输出标准审查意见（5分钟/份）

## 为什么基于流程体系？

- **稳定性**：人会流动、岗位会调整，但流程的演变是渐进式的
- **完整性**：流程的每个节点都是显性的，缺失一目了然
- **可组装**：基于流程节点拆解的原子Skill可被任意角色按需组装
- **可度量**：流程节点有明确的输入、输出和质量标准
